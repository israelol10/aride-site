Soluciones que estamos construyendo

El objetivo de ARIDE es evolucionar hacia una biblioteca de productos inteligente, capaz de sugerir configuraciones, detectar errores y documentar cambios de ingeniería.

Biblioteca técnica de productos

Estructuramos componentes y sistemas (como el axial fan) con vistas “caja negra” y “arquitectura” que luego pueden ser explotadas por IA.

  • Definición de funciones, puertos y requisitos
  • Relación entre ítems, mates y cadenas funcionales
  • Formato estándar: YAML / JSON

Asistentes técnicos privados

Chats internos que entienden especificaciones, índices de búsqueda o documentación de proyectos y responden con foco técnico.

  • Conexión a índices / bases de datos
  • Respuestas soportadas por fuente documental
  • Uso controlado de modelos de lenguaje

Flujos de diseño y verificación

Soporte para procesos de desarrollo: desde la idea del producto hasta su liberación, pasando por revisión de requisitos y variantes.

  • Checklists guiadas con IA
  • Resúmenes automáticos de cambios
  • Soporte a documentación de ingeniería

Casos de uso y enfoque actual

ARIDE está orientado a entornos donde conviven diseño mecánico, documentación técnica extensa y necesidad de automatizar conocimiento.

Ventiladores axiales y subsistemas

Ejemplo de producto modelado con vistas “black box” y “architecture view”, incluyendo funciones, puertos, mates y cadenas funcionales.

  • Modelo funcional y geométrico del fan
  • Definición de interfaces y requerimientos
  • Base para generar variantes futuras

Conveyors y manejo de materiales

Objetivo a mediano plazo: estructurar una biblioteca para sistemas de transporte, siguiendo el mismo enfoque arquitectónico.

  • Identificación de módulos estándar
  • Plantillas de puertos mecánicos y de potencia
  • Documentación lista para IA

Boilers y sistemas térmicos

Exploración de cómo representar calderas y sistemas asociados usando el mismo lenguaje: funciones, puertos, balances y arquitectura.

  • Requisitos térmicos y de seguridad
  • Balance de energía, masas y estados
  • Soporte para selección y comparación
